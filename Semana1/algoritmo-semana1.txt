Cómo compartir información confiable en internet.

Entrar a internet
Buscar la pagina web del correo electronico
Loging con el usuario y contraseña correspondiente
Buscar en el menu la opción de interes(enviar el correo)
git addSubir el documento PDF para enviar
Colocar el mensaje que acompañe el documento
git tag Buscar el destinatario del correo
git verify-commit Verificar que se envie la información completa documento y mensaje
git push Enviar el correo al receptor
